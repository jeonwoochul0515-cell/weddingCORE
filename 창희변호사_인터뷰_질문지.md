# 창희 변호사 인터뷰 질문지 (weddingCORE 룰 코드화)

**목적**: Phase 1 A 기능(컴플라이언스 체크리스트 타임라인)의 `obligationTemplates` 40+ 항목을 확정하기 위한 법률 전문가 인터뷰.
**예상 소요**: 2~3회 미팅, 회당 90분.
**사전 공유 자료**: 본 문서, Firestore 스키마 문서, 연구 문서(`국제결혼_법률AI_플랫폼_연구.md`).

---

## 인터뷰 구조

- **Round 1 (법률 검토)**: 40+ 의무 항목의 법적 근거·기한·증빙·위반 시 제재 확정
- **Round 2 (업무 플로우 매핑)**: 각 항목의 실제 업체 수행 방식, 자주 누락되는 지점
- **Round 3 (판례·실무 사례)**: 실제 행정처분·판결 사례로 본 리스크 우선순위

## Round 1 — 법률 검토 질문

### 1. 결혼중개업법 핵심 의무

#### 1-1. 등록·갱신·변경 관련 (Stage 1: 계약 전)
- 국내 / 국제 결혼중개업 등록 요건 차이는? 자본금·보증보험 기준은?
- 등록증 갱신 주기와 갱신 실패 시 법적 효과는?
- 상호·대표자·소재지 변경 시 신고 기한은? (제4조, 제5조)
- 신고 누락 시 과태료 범위는?

#### 1-2. 광고 규제 (Stage 1)
- 결혼중개업법 §12 광고 금지사항 전체 리스트는?
- "국적 특정 광고", "허위·과장 광고"의 판단 기준 (실제 사례)?
- 온라인 광고(블로그, 유튜브 포함) 적용 범위는?
- 광고 위반 발생 시 행정처분 기준(경고 → 영업정지 → 등록취소)?

#### 1-3. 계약 관련 (Stage 2: 계약 체결)
- 서면계약 의무의 정확한 범위 (§10)?
- 필수 기재사항 체크리스트 (전액, 환불조건, 배상책임, 계약기간 등)?
- 표준약관과 개별약관의 우선순위는?
- 소비자분쟁해결기준상 환불 조건의 구체적 단계별 기준?
- 불공정 약관으로 자주 적발되는 조항 Top 5는?

#### 1-4. 신상정보 제공 의무 (Stage 3: 맞선 준비) ★ 행정처분 최다 항목
- §10의2 상호제공 의무 9개 항목의 정확한 법조문 리스트:
  1. 혼인경력, 2. 직업, 3. 건강상태(질병·장애), 4. 범죄경력, 5. 학력, 6. 현 거주지, 7. 가족관계, 8. 재산, 9. 기타?
- "모국어 번역 제공" 의무의 판례상 해석 (요약본 가능? 전문 필수?)
- 수령 확인 서면의 법적 효력 (전자서명 허용? 타임스탬프 필요?)
- 허위 신상정보 제공 시 업체의 책임 범위 (주의의무 vs 확인의무)?
- 실제 "신상정보 미제공"으로 행정처분 받은 사례의 공통 패턴?

#### 1-5. 맞선·혼인 (Stage 4)
- 현지법 혼인요건 확인은 누구의 의무인가 (업체 vs 당사자)?
- 혼인신고 절차 지원의 법적 한계 (행정사 업무와의 경계)?
- 맞선 후 혼인 성사 실패 시 업체의 의무는?

### 2. 출입국관리법 / F-6 비자 (Stage 5)

- F-6 비자 3개 유형(F-6-1, F-6-2, F-6-3)별 요건 차이?
- 소득요건 연도별 기준 (가구원수별 연소득)?
- 재산 환산 공식의 실무 기준?
- 의사소통 요건(한국어 능력 or 모국어 능력)의 허용 범위?
- "혼인 진정성" 심사 시 중요시되는 증빙 자료 Top 10?
- 결혼이민사전안내프로그램 이수 의무 (누구·언제·어디서)?
- 비자 불허 시 재신청 전 반드시 해결해야 할 사항은?

### 3. 사후관리 (Stage 6)

- 결혼중개업법상 사후관리 의무의 법적 근거와 기간?
- 사후관리 이행 증빙으로 법원·행정청이 인정하는 수준?
- 다문화가족지원센터 연계의 법적 성격 (권고 vs 의무)?

### 4. 국제사법 · 기타

- 외국인 배우자의 본국법 적용 쟁점 (혼인 요건, 혼인무효 사유)?
- 혼인무효 vs 이혼의 실무 판단 기준 (사기결혼 관점)?
- 사기결혼 형사고소 시 피해자(한국인)가 입증해야 할 요건?

## Round 2 — 업무 플로우 매핑 질문

### 각 의무 항목별 공통 질문

아래 4가지 질문을 40+ 항목 각각에 대해 확인:

1. **Trigger**: 언제 이 의무가 시작되는가? (특정 이벤트 또는 타이밍)
2. **Due Date 계산 공식**: "OO로부터 N일 이내" 등 정량 기준
3. **증빙**: 이행을 입증하는 서류·기록의 최소 수준
4. **위반 시 제재**: 과태료 / 영업정지 / 등록취소 / 형사처벌 중 어느 것?

### 업체 현장 질문 (실무 소장 경험 기반)

- 영세 업체가 가장 많이 놓치는 의무 Top 3?
- 매니저 혼자 처리하는 업체의 공통 취약점은?
- "서류는 있지만 법적 효력이 없는" 사례 패턴은?
- 외국인 배우자 측 자료 확보 시 자주 생기는 문제?
- 에이전시(해외 현지 중개인)와의 협업에서 법적 리스크 발생 지점?

## Round 3 — 판례·실무 사례 질문

### 개별 사례 심층 인터뷰

연구 문서에 언급된 대표 사례들을 기반으로 구체적 룰화 포인트 도출:

#### 3-1. 수원지법 평택지원 8,000만원 배상 판결
- 판결문 원문 공유 가능?
- 이 사안의 핵심 쟁점 3가지?
- 이 쟁점을 AI가 사전 감지하려면 어떤 체크가 필요?

#### 3-2. 수원지법 안산지원 사기결혼 형사처벌 사례
- 사기결혼 입증의 결정적 증거는?
- AI가 사기결혼 의심 패턴을 감지할 수 있는 선행 지표?

#### 3-3. 광주지법 월 30건 혼인무효소송 사례
- 혼인무효 인용 vs 기각 판결의 분기점?
- 업체가 소송 리스크를 최소화하려면 어느 단계에서 어떤 조치를?

#### 3-4. 대법원 2020므15896 전원합의체
- 이 판례가 결혼중개업에 미친 실무적 변화?
- 기존 관행 중 바꿔야 할 사항?

### 시나리오 기반 질문

다음 시나리오에서 업체·고객·변호사가 해야 할 조치는?

1. 외국인 배우자가 입국 후 2개월 만에 가출 + 연락두절
2. 외국인 배우자가 비자 발급 전에 신청을 철회
3. 한국인 배우자가 혼인신고 후 신상정보 허위를 주장하며 환불 요구
4. 업체가 신상정보 번역을 요약본만 제공 후 고객이 행정처분 제기
5. F-6 비자 불허 사유가 "진정성 부족"일 때 재신청 전 필수 조치

## 부록 A: 룰 코드화 템플릿 (각 의무 항목당 작성)

창희 변호사님께서 각 의무 항목을 아래 템플릿으로 작성해 주시면
Firestore `obligationTemplates` 컬렉션에 그대로 입력됩니다.

```
[의무코드] BG_INFO_EXCHANGE
[단계] 3 (맞선 준비)
[분류] pre_meeting
[제목] 신상정보 상호 제공
[설명] 양측의 혼인경력·직업·건강·범죄경력 등 9개 항목을
       서면으로 상호 제공하고 모국어 번역 및 수령 확인을 받아야 함.
[법적근거] 결혼중개업법 제10조의2, 동법 시행령 제3조의2
[제재] 1차: 경고 / 2차: 영업정지 1개월 / 3차: 영업정지 3개월
       과태료 최대 1,000만원 (결혼중개업법 제27조)
[기한규칙]
  - 유형: relative
  - 기준점: meeting_scheduled (맞선 예정일)
  - 오프셋: -7 (맞선 7일 전까지 완료)
[필수증빙]
  - 혼인관계증명서 (유효 3개월)
  - 건강진단서 (유효 6개월)
  - 범죄경력증명서 (유효 6개월)
  - 재직증명서 또는 소득증명 (유효 3개월)
  - 학력증명서
  - 가족관계증명서 (유효 3개월)
  - 주민등록등본 (유효 3개월)
  - 재산증명 자료
  - 기타 약정 항목
[체크룰]
  - 유형: translation_completed
  - 파라미터: { "requiredLanguages": ["ko", "vi"],
                "requiredDocuments": 9 }
[중요도] critical
[순서] 301
```

## 부록 B: 예상 40+ 의무 항목 초안 리스트

인터뷰 전 예비 목록. Round 1 후 확정·가감.

### Stage 1: 계약 전 (5개)
1. 등록증 유효성 확인
2. 등록 정보 변경 신고 (상호·대표·소재지)
3. 광고 문구 적법성 검토
4. 소비자 사전 안내 이행
5. 사업장 내 등록증 게시

### Stage 2: 계약 체결 (6개)
6. 서면계약 체결
7. 환불조건 명시
8. 배상책임 명시
9. 계약기간·중개범위 명시
10. 표준약관 사용 여부 확인
11. 불공정 약관 제거

### Stage 3: 맞선 준비 (9개) ★ 최다 위반 단계
12. 신상정보 9개 항목 수집 (한국인 측)
13. 신상정보 9개 항목 수집 (외국인 측)
14. 범죄경력증명서 확보 (양측)
15. 건강진단서 확보 (양측)
16. 혼인관계증명서 확보 (양측)
17. 아동학대 범죄 이력 공개 확인
18. 모국어 번역본 생성
19. 상호 서면 제공 및 수령 확인
20. 검증완료 증명 기록 보관

### Stage 4: 맞선·혼인 (6개)
21. 맞선 일정·장소 기록
22. 통역 제공
23. 현지법 혼인요건 확인
24. 혼인신고 서류 안내
25. 혼인신고 접수 증빙 확보
26. 혼인 성사 여부 기록

### Stage 5: 비자 (8개)
27. F-6 소득요건 체크 (가구원수 기준)
28. 주거요건 확인
29. 진정성 증빙 자료 수집
30. 결혼이민사전안내프로그램 이수 확인
31. 서류 유효기간 관리 (20종)
32. 비자 신청 접수 증빙
33. 불허 사유 기록 및 대응
34. 재신청 전 보완 조치

### Stage 6: 사후관리 (6개)
35. 입국 1개월 체크
36. 입국 3개월 체크
37. 입국 6개월 체크
38. 입국 12개월 체크
39. 다문화가족지원센터 연계 안내
40. 사후관리 이력 서면 보관
