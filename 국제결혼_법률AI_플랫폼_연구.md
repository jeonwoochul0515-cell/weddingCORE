# 국제결혼중개업 플랫폼: 법률서비스 + AI 통합 모델 연구

**2026년 4월 | 우철 × 창희 변호사 협업 구상**

---

## 1. 왜 법률+AI인가: 시장의 구조적 법률 수요

국제결혼은 처음부터 끝까지 법률 이벤트의 연속입니다.

### 1.1 국제결혼 생애주기별 법률 이슈

| 단계 | 법률 이슈 | 현재 해결 방식 | 문제점 |
|------|----------|-------------|--------|
| **계약 전** | 업체 등록 적법성, 광고 적법성 | 소비자가 직접 확인 (34%만 공시 확인) | 66%가 미확인 |
| **계약** | 서면계약 의무, 환불조건, 배상책임 | 업체 자체 약관 (불공정 약관 다수) | 소비자 불리 |
| **맞선 전** | 신상정보 상호 서면제공 의무, 모국어 번역 | 수작업, 형식적 이행 | 행정처분 최다 사유 |
| **맞선/결혼** | 현지법 혼인 요건, 혼인신고 | 업체/행정사 임의 처리 | 절차 누락 빈번 |
| **비자** | F-6 소득요건, 진정성 심사, 안내프로그램 이수 | 행정사 외주 또는 자력 | 불허 시 대응 못함 |
| **결혼생활** | 가정폭력, 의사소통 갈등, 체류자격 문제 | 다문화가족지원센터 (제한적) | 법률 접근성 낮음 |
| **분쟁 발생** | 사기결혼, 가출, 이혼, 혼인무효, 양육권, 위자료 | 변호사 개별 상담 (비용 부담) | 건당 수백만원 |
| **업체 분쟁** | 환불 거부, 허위정보, 손해배상 | 소비자원 상담 (실효성 낮음) | 업체 귀책 입증 어려움 |

### 1.2 핵심 데이터

- 혼인 지속율 90.7% → 나머지 9.3%는 이혼(5.4%), 가출(2.9%), 별거(0.9%)
- 연간 국제결혼 약 2만건 × 9.3% = 연간 약 1,860건의 법적 분쟁 잠재 건수
- 혼인무효소송: 한 지방법원에서만 월 30건 접수 (광주지법 사례)
- 업체 대상 손해배상 판결: 최대 8,000만원 (수원지법 평택지원)
- 사기결혼 형사처벌: 징역 6개월/집행유예 2년 (수원지법 안산지원)

## 2. 법률+AI 통합 플랫폼 설계

### 2.1 제품 컨셉

**"국제결혼의 모든 법적 리스크를 AI가 사전에 차단하고, 문제 발생 시 변호사가 즉시 대응하는 플랫폼"**

기존 LAW-CADDY가 변호사를 위한 도구라면, 이 플랫폼은:
- 업체에게는 "법 위반 방지 자동화 시스템"
- 고객에게는 "국제결혼 법률 안전망"
- 변호사(창희 변호사님)에게는 "사건 유입 채널"

### 2.2 AI 기능 설계 (6개 AI 에이전트)

#### AI Agent 1: 컴플라이언스 가디언
- 업체가 수행하는 모든 단계에서 법적 의무사항 자동 체크
- 신상정보 제공 누락 시 실시간 경고
- 광고 문구 작성 시 결혼중개업법 제12조 위반 여부 자동 검토
- 공시 데이터 자동 생성 (성평등가족부 양식에 맞게)
- 행정처분 이력 모니터링 및 알림

#### AI Agent 2: 계약서 분석기
- 업체의 계약서를 AI가 분석하여 불공정 약관 자동 검출
- 소비자분쟁해결기준 대비 환불조건 적정성 평가
- 표준약관과 차이점 하이라이트
- 모국어 번역본 자동 생성 (베트남어, 캄보디아어, 중국어 등)
- 계약 체결 시 양측 서명 + 타임스탬프 기록 (증거 보존)

#### AI Agent 3: 신상정보 검증기
- 양측 신상정보 교환 의무 항목 체크리스트 자동 관리
- 제출된 서류(범죄경력, 건강진단, 혼인관계증명 등)의 유효기간 자동 확인
- 아동학대 범죄 이력 공개 확인
- 모국어 번역 자동 생성 및 제공 기록 보관
- 정보 제공 완료 시 "검증완료" 디지털 증명서 발행

#### AI Agent 4: 비자 어시스턴트
- F-6 비자 신청 요건 자동 체크 (소득, 의사소통, 진정성)
- 필요 서류 체크리스트 자동 생성 (20종+)
- 소득요건 계산기 (가구원수별 기준, 재산 환산)
- 서류 유효기간 만료 경고
- 비자 불허 시 대응 가이드 및 변호사 연결

#### AI Agent 5: 분쟁 예방/대응 어드바이저
- 결혼 생활 중 법적 분쟁 징후 감지 시 조기 경고
- 가출/연락두절 시 대응 프로토콜 자동 안내
  - 출입국 사실조회 신청 가이드
  - 공시송달 절차 안내
  - 혼인무효 vs 이혼 판단 기준 제공
- 업체 대상 환불/손해배상 청구 시 필요 증거 가이드
- 변호사 상담 사전 정보 정리 (AI가 사건 요약서 자동 생성)

#### AI Agent 6: 다국어 커뮤니케이터
- 한국어 ↔ 베트남어/캄보디아어/중국어/우즈베크어 실시간 번역
- 법률 용어의 정확한 모국어 번역 (일반 번역기와 차별점)
- 외국인 배우자 권리 고지서 자동 생성
- 다문화가족지원센터 연계 안내 (모국어)

### 2.3 법률서비스 통합 모델 (창희 변호사님 역할)

#### 티어 1: AI 자동 법률 서비스 (무료/SaaS 포함)
- 컴플라이언스 자동 체크
- 계약서 자동 분석
- 비자 서류 체크리스트
- 법률 FAQ 챗봇

#### 티어 2: AI + 변호사 하이브리드 (유료)
- AI가 사건 사전 분석 → 변호사가 검토/확정
- 계약서 법률 검토 (AI 초안 + 변호사 확인)
- 비자 불허 이의신청 자문
- 가격: 건당 10~30만원

#### 티어 3: 변호사 직접 대응 (유료)
- 혼인무효소송, 국제이혼소송
- 업체 대상 손해배상소송
- 사기결혼 형사고소
- 가정폭력 관련 법률 대응
- 가격: 착수금 100~300만원

## 3. 수익 모델 확장

### 3.1 기존 SaaS 수익 + 법률 수익

## 4. 경쟁 해자 (MOAT) 분석

### 4.1 왜 따라오기 어려운가

| 해자 | 설명 |
|------|------|
| **법률 전문성** | 창희 변호사님 — 가족법 전문, 동아대 겸임교수, 실제 소송 경험. 일반 개발자가 법률 로직을 구현하기 극히 어려움 |
| **AI + 법률 데이터** | LAW-CADDY에서 축적된 RAG 아키텍처, 14만건 판례 DB, 법제처 API 연동 노하우를 그대로 이식 |
| **결혼중개업법 특화** | 결혼중개업법, 국제사법, 출입국관리법, 다문화가족지원법의 교차점을 아는 팀은 사실상 없음 |
| **규제와 동행** | 규제 강화 = 플랫폼 수요 증가. 규제가 해자가 되는 구조 |
| **네트워크 효과** | 업체가 많이 쓸수록 → 법률 데이터 축적 → AI 정확도 향상 → 더 많은 업체 유입 |

## 5. 구체적 시나리오: AI가 어떻게 작동하는가

### 시나리오 1: 업체 매니저의 일상 업무

```
08:30 — 매니저가 플랫폼에 로그인
  AI 알림: "고객 김OO의 비자 서류 중 소득금액증명원 유효기간이 
  7일 후 만료됩니다. 갱신을 안내해 주세요."

09:00 — 새 고객 상담 접수
  AI: 계약서 자동 생성 (고객 조건 기반 맞춤)
  AI: "이 고객은 가구원 3인이므로 F-6 소득요건이 연 OO만원입니다.
  현재 소득으로는 충족됩니다."

10:00 — 베트남 에이전트가 후보자 3명 프로필 등록
  AI: "후보자 B의 혼인관계증명서가 누락되었습니다. 
  결혼중개업법 제10조의2에 따라 맞선 전까지 반드시 확보해야 합니다."

14:00 — 고객에게 프로필 전달 준비
  AI: 신상정보 제공 기록 자동 생성
  AI: 베트남어 번역본 자동 생성
  AI: "양측 신상정보 교환 완료. 디지털 증명서가 발행되었습니다."

16:00 — 기존 고객 이OO의 배우자 입국 후 3개월 체크
  AI: "사후관리 3개월 체크 시점입니다. 
  가까운 다문화가족지원센터: 부산 연제구센터 (051-XXX-XXXX)"
```

### 시나리오 2: 고객이 사기결혼 피해를 당했을 때

```
고객: "아내가 입국 2개월 만에 집을 나갔습니다. 연락이 안 됩니다."

AI Agent 5 (분쟁 대응):
1. "출입국 사실조회를 먼저 신청하세요. [신청서 자동 생성]"
2. "현 상황은 '악의적 유기'에 해당할 수 있습니다."
3. "혼인무효소송과 이혼소송 중 선택이 필요합니다:
   - 혼인무효: 처음부터 부부가 아니었던 것으로 (사기결혼 입증 필요)
   - 이혼: 혼인 해소 (위자료 청구 가능)
   대법원 2020므15896 전원합의체에 따르면..."
4. "법률 상담을 원하시면 전문 변호사에게 연결해 드립니다.
   [AI가 사건 요약서 자동 생성 → 변호사에게 전달]"
5. 변호사 상담료: 초회 무료 (플랫폼 회원), 이후 건당 20만원
```

### 시나리오 3: 비자 불허 대응

```

AI Agent 4 (비자 어시스턴트):
1. "불허 사유를 확인해 주세요: [체크리스트 제공]"

   - 재산소득 환산 방법: 순자산 × 5% ÷ 12개월
   - 동거 직계가족 소득 합산 가능"
3. "진정성 부족인 경우:
   - 추가 교제 증빙 필요 (사진 20장+, 통화내역, SNS)
   - 6개월 후 재신청 가능"
4. "법적 이의가 필요한 경우 행정사/변호사 연결"
```

## 6. 창희 변호사님 협업 구조

### 6.1 역할 분담

| 역할 | 우철 | 창희 변호사 |
|------|------|-----------|
| 플랫폼 개발 | 풀스택 개발, AI 연동 | - |
| 법률 로직 설계 | - | 결혼중개업법/가족법/국제사법 기준 |
| AI 프롬프트/데이터 | RAG 아키텍처, 벡터DB | 법률 Q&A 데이터셋, 판례 분석 |
| 계약서 템플릿 | UI 구현 | 표준약관 기반 법률 검토 |
| 법률 상담 | AI 사전분석 | 실제 상담 및 소송 수행 |
| 영업 | 업체 온보딩, 기술 지원 | 법률 세미나, 업계 신뢰도 |

### 6.2 수익 배분 (안)

| 서비스 | 수익 배분 |
|--------|----------|
| SaaS 구독료 | 플랫폼 100% (개발/운영비) |
| AI 법률 분석 (자동) | 플랫폼 100% |
| AI+변호사 하이브리드 | 플랫폼 30% + 변호사 70% |
| 소송 수임 | 법률사무소 청송 100% (플랫폼은 리드 제공) |
| 법률 콘텐츠/세미나 | 50:50 |

### 6.3 법률사무소 청송의 확장 가치

- 현재 전문분야: 프랜차이즈법, 가족법, 학교폭력, 형사/행정
- 국제결혼 분야 추가 시: 가족법 + 국제사법 + 다문화 영역 확장
- 부산 연제구 소재 → 부산/경남 국제결혼 업체 접근성 좋음
- 동아대 겸임교수 → 학술/정책 영향력 활용 가능

## 8. 결론: 이 사업의 본질

일반적인 SaaS는 "업무를 편리하게"가 가치 제안입니다.
이 플랫폼은 **"법적 리스크를 제거하고, 문제 발생 시 즉시 대응"**이 가치 제안입니다.

442개 영세 업체에게 "편리함"은 nice-to-have지만,
**"행정처분 안 당하는 것"과 "소송에서 지지 않는 것"은 must-have**입니다.

그리고 이 must-have를 제공할 수 있는 팀은:
- 법률 전문가 (창희 변호사) + 개발자 (우철) + AI (Claude/LAW-CADDY 노하우)

이 조합을 가진 경쟁자는 현재 제로입니다.
