# weddingCORE 가격 정책 설계

> 작성일: 2026-04-17
> 근거: 시장 리서치 + 누나 인터뷰 + CODEF API 보유 + 15개국 법률 데이터

---

## 1. 핵심 전제

### 시장 현실

| 팩트 | 출처 |
|------|------|
| 전국 등록 업소: **~400개** | 여성가족부 공시 |
| 업소는 **월 5만 원 이상 안 냄** | 누나 직접 답변 |
| 남성 고객이 쓰는 돈: **1,372만 원** (중개수수료) | 여성가족부 실태조사 |
| 행정사 서류 대행비: **50~150만 원** | 업계 시세 |
| 연간 국제결혼: **~21,000건** | 통계청 2024 |
| CODEF API 17개: **이미 보유** | LawDocs/law-caddy |

### 가격 설계 원칙

1. **SaaS 구독은 미끼, 서류 자동화가 본체** (Toast 모델)
2. **남성 고객에게 직접 과금** (돈이 있는 쪽에서 받는다)
3. **무료 → 유료 전환 퍼널** (법률 정보 무료 → 서류 패키지 유료)
4. **행정사 반값** (가격 앵커: 행정사 100만 원 → 우리 30~50만 원)
5. **업소는 저가로 붙잡기** (리텐션 도구, 수익원 아님)

### 2.2 중개업소 (B2B) — 리텐션 도구

| 플랜 | 월 가격 | 포함 기능 | 타겟 |
|------|---------|----------|------|
| **Free** | **0원** | 업소 프로필 등록, 리드 수신 3건/월 | 체험 |
| **Basic** | **3만 원** | CRM(고객30명), 매칭도구(MBTI/사주), 리드 수신 10건/월 | 신규/소규모 업소 |
| **Standard** | **5만 원** | Basic + 컴플라이언스 체크리스트, 규제 레이더, 리드 무제한, 고객 서류패키지 대행 발주 | 활발한 업소 |

#### 업소가 서류 패키지를 대행 발주할 때

```
남성 고객 → 업소에 의뢰 → 업소가 weddingCORE에서 발주
                         → 고객이 공인인증서 인증
                         → 서류 자동 수집
                         → 업소 대시보드에서 확인

업소 결제: 건당 15만 원 (도매가)
고객 청구: 업소가 자유롭게 (30~50만 원 등)
업소 마진: 건당 15~35만 원
```

→ 업소 입장에서 **서류 대행이 새로운 수익원**이 됨

## 4. 수익 시뮬레이션

### 4.1 시나리오별 월 수익

| 시나리오 | B2C 서류 패키지 | B2B SaaS | 법률 상담 | 합계 |
|---------|---------------|----------|----------|------|
| **초기 (6개월차)** | 30건 × 25만 = 750만 | 20곳 × 4만 = 80만 | 10건 × 5만 × 30% = 15만 | **~845만/월** |
| **성장 (12개월차)** | 100건 × 30만 = 3,000만 | 50곳 × 5만 = 250만 | 30건 × 5만 × 30% = 45만 | **~3,295만/월** |
| **확장 (24개월차)** | 300건 × 30만 = 9,000만 | 100곳 × 5만 = 500만 | 60건 × 7만 × 30% = 126만 | **~9,626만/월** |

### 4.2 연간 환산

| 시점 | 연 매출 | 핵심 수익원 비중 |
|------|---------|----------------|
| 1년차 | **~1억 원** | 서류 89% / SaaS 9% / 법률 2% |
| 2년차 | **~4억 원** | 서류 91% / SaaS 8% / 법률 1% |
| 3년차 | **~12억 원** | 서류 93% / SaaS 5% / 법률 2% |

→ **서류 자동화가 매출의 90%**. SaaS는 리텐션/채널 역할.

### 4.3 시장 침투 가능성 검증

```
연간 국제결혼 ~21,000건
  × F-6 비자 신청 비율 ~70% = ~14,700건
  × 서류 대행 수요 ~30% = ~4,400건/년 (잠재 시장)
  × weddingCORE 점유 5% = ~220건/년 (초기 목표)
  × 평균 단가 30만 원 = 연 6,600만 원 (서류 패키지만)

현실적 1년차 목표: 월 30~50건 × 30만 = 연 1~1.8억
```

```

│         → 월 매출: 25건 × 7만 × 30% = 52.5만
│
└─ 10% → 업소 연결 리드 (200명)
          → 업소 SaaS 구독 + 서류 대행 도매
```
